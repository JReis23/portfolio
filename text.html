Bonjour à tous. 


Comme vous pouvez le constater en parcourant mon site web, il y a plusieurs choses qui ne sont pas terminées. 


Ceci est volontaire, car mon objectif est de documenter ici dans le blog les différentes étapes qui seront faites pour résoudre les problèmes, ou simplement pour mettre en ligne de nouvelles fonctionnalités. 


Ce site est réalisé en Svelte, en utilisant Sveltekit, et Supabase pour le backend. 


Vous pouvez laisser des commentaires sur ce que vous pensez être mauvais, ce que vous aimez ou simplement ce que vous aimeriez voir mis en œuvre. Laissez vos idées afin que nous puissions, ensemble, faire évoluer ce site web et éventuellement créer quelque chose d'intéressant.


Les premiers pas pour le Blog :


  • Faire apparaître les erreurs quand elles existent
  • Créer la possibilité de modifier les messages créés
  • Commencez à utiliser TypeScript avec Svelte


N'hésitez pas à visiter mon GitHub pour voir l'évolution du code.
